# 菩提思

个人闲暇时开发的网站，主要包含：文章、工具等内容。

## 环境
* nodejs v18.18.0
* yarn v1.22.11
* IDE - VsCode:
  * Vue - Official：Vue3高亮展示等
  * EditorConfig for VS Code：编辑器规范
  * ESLint：代码规范
  * Prettier - Code Formatter：代码格式化

## 安装
```
yarn install
```

## 运行

```
yarn dev
```

## 编译

```
yarn build
```

## 技术栈

- 前端
  - Vue
  - Vite
  - TS & JS
  - Less
  - ant-design-vue
- 后端
  - java
