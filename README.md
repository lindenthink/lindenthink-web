# 菩提思

个人闲暇时开发的个人网站，主要包含：文章、工具等内容。

## 编译

```
yarn build
```

## 运行

```
yarn dev
```

## 采用技术栈

- 前端
  - Vue
  - Vite
  - TS & JS
  - Less
  - ant-design-vue
- 后端
  - java
